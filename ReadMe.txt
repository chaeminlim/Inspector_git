Inspector 설명

1- get Processes 버튼을 누르면 현재 프로세스 정보를 가져와서 왼쪽 트리 뷰에 뿌려줍니다.

2- 트리 뷰 목록중 자세히 보고싶은 것을 클릭하면 상세 정보를 보여줍니다.

3- 아마 클릭할때 화면에 하얀색으로 사각형이 그려질텐데 selector 기능에 추가할 건데 잠시 옮겨놓은 것입니다.

4- selector 기능은 마우스 후킹으로 마우스 글로벌 위치를 얻을 수 있으면 selector를 구현할 수 있습니다.
$$$$(마우스 후킹을 못하고 있습니다 ㅜ)$$$$

5- call handler 버튼은 트리뷰에서 선택한 automationElement를 분석해서
  호출할 수 있는 기능을 오른쪽 리스트 박스에 넣을 예정인데 그부분은 하드코딩영역이라 이름만 넣어놓도록 잠시 설정했습니다.

6-  로딩시간이 조금 걸리는데 재귀함수로 구현해서 그런거라서 나중에 모든 기능이 구현되면 수정할 예정입니다.
